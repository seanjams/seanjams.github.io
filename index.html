Key Wheel
Demo • Repo

They Key Wheel is a visualization of 7-tone musical scales that I discovered while doing mathematical research at UC Davis. The network of scales can be used by musicians to find cool new ways to change keys, and see which keys contain certain chords. So far, the Key Wheel has helped me find really interesting and non-obvious cycles of notes that can be used as jazz progressions.

Scalar
Demo • Repo

Scalar is a web application designed for guitarists and pianists. These two common instruments have really different visualizations, and sometimes it can be difficult to quickly transcribe a song on one of these instruments to the other. With Scalar, as the user selects notes, they are sounded and highlighted accross both instruments in all possible places. A pattern selector allows the guitar to be divided into slices of notes to see inversions accross the fretboard.

NodeItAll
Demo • Repo

NodeItAll is a QA site for tech related questions inspired by StackOverflow. Users can create an account, ask and answer questions, search for keywords, and vote up or down on all items. In each question/answer form, the user can also input syntax highlighted code in a variety of languages to better illustrate their problems/solutions. Built with React/Redux and Ruby/Rails.

PlaceFinder
Demo • Repo

PlaceFinder is a client-side web application that utilizes Google Maps and Places API's to display nearby locations that match a keyword search. It is implemented in both vanilla Javascript and React.js to exhibit different design patterns. Search can be filtered by popularity or distance, and results repopuate after user scrolls too far on the map. Enable geolocation.

CityImpact
Demo • Repo

CityImpact is a mobile application where users can get more involved in their communites by hosting events ranging from park/neighborhood clean-ups to block parties. The app is being built in collaboration, using React Native and Node.js. My job has been constructing and maintaining the backend, designing the schema and implementing Facebook and Google login with OAuth and Passport.js. After logging in, users can create, discover, and join events by interacting with the map.
